Le comportement de l'intelligence artificielle dans les jeux vidéos

Publié le 28 Octobre 2019 à 19h20

Illustration tirée du blog archimag

Pour qu'une IA soit réussite dans un jeu vidéo, elle doit bien s'incruster dans le monde, sa façon de se comporter doit coller avec le monde qui l'entoure. Si elle fait n'importe quoi, l'experience du joueur peut être dégradé. C'est la conséquence d'un "bug".

LES BUGS

C'est une anomalie dans un fonctionnement informatique. Celui-ci peut être dû à une contradiction logique dans la programmation de l'intelligence.

Personnage non joueur, dans le jeu Final Fantasy XV, en plein bug, assis sur le vide

C'est pour ça qu'il faut qu'une intelligence artificielle bien codé. L'algorithme doit être bien pensé. Nous allons passer en revu les differentes technique utilisé pour maitriser l'ia dans les jeux-vidéo.

On peut reconnaitre differente type d'IA dans les jeux vidéos. Il y a les IA amicales, celles qui coopère avec le joueur pour l'aider à gagner et d'autre IA qui sont les ennemies du joueurs, les agents. Ces agents s'adapte en fonction des choix du joueur, et peu communiquer avec d'autres agents.

LA RECHERCHE DE CHEMIN

En anglais, le pathfinding, c'est la recherche du meilleur chemin en partant d'un point A à un point B, tout en respectant certaines contraites.

Pour le pathfinding, on utilise généralement l’algorithme A*. Une version améliorée, plus rapide, de l’algorithme de Dijkstra. Celui-ci ayant pour but, dans un plan 2D d’un jeu vidéo par exemple, de parcourir dans l’ordre (en fonction de la distance qui sépare le point de départ et le point d’arrivé) toutes les cases possibles, jusqu’à trouver la case d’arrivée.

Chemin trouvé par l'algorithme Dijkstra. Image tirée du site Semantic scholar

LES COMBATS

Dans cette vidéo, nous pouvons voir 2 robots se battre en duel sur le jeu Super Smash Bros. Les deux robots réalisent les mêmes coups au même moment car leur algorithme est le même. On peut en conclure qu'une intelligence artificielle se doit d'avoir un algorithme qui envisage toutes les possibilités, même celles où l'adversaire a les mêmes mouvements que lui-même.
